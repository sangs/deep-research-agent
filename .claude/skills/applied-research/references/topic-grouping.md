# Topic Grouping Rules

## Label Format

- 2–6 words, title case
- Named after entity / event / product / theme
- **No verbs** (not "OpenAI Releases GPT-5" → use "OpenAI GPT-5 Release")
- Examples: `OpenAI GPT-5`, `EU AI Act`, `Apple WWDC 2026`, `Nvidia H200 GPU`

## Clustering Rules

- Group by the primary subject: company, product, event, regulation, or concept
- One cluster per distinct entity/event even if angles differ
- Same story from 2+ sources → separate cards within ONE cluster (no dedup)
- Generic catch-all clusters (e.g. "Miscellaneous") are acceptable for outlier articles

## Sort Order

Topics sorted by `article_count` descending (most-covered story first).

## No Deduplication

**Never remove duplicate stories.** The same article appearing in multiple sources
is displayed as multiple cards in the same topic cluster. Diversity of coverage is a feature.
